# SMF

**This repo is not production-ready yet. I am still developing the core features.**

The SMF (Session Management Function) is a core component of the 5G architecture. It is responsible for PDU session management, IP address allocation, and coordination with the UPF for user plane traffic handling.

`smf-rust` is a Rust implementation of 3GPP's 5G SMF specification. The specification can be found [here](https://portal.3gpp.org/desktopmodules/Specifications/SpecificationDetails.aspx?specificationId=3334).
